DSD generator generates several pattern and frequencies. Use type XMC_DSD_GENERATOR_CONFIG_t for this data structure. Note: Output pins have to be enabled by user.

Dokumentation der Datenelemente

uint32_t bit_reverse

should PWM signal be bit-reversed? 0: Normal mode, 1:Bit-reverse mode

uint32_t frequency

Frequency divider value of PWM signal. This parameter can take a value of XMC_DSD_GENERATOR_CLKDIV_t

uint32_t generator_conf

Carrier generator configuration register(CGCFG)

uint32_t inverted_polarity

should PWM signal start from negative max. 0: Normal, 1: Inverted

uint32_t mode

carrier generator operating mode. This parameter can take a value of XMC_DSD_GENERATOR_MODE_t

Wird benutzt von XMC_DSD_Generator_Start().
