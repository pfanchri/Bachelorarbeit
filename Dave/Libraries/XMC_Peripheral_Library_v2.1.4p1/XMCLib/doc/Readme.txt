XMCLib documenation
===================

To generate the documenation the path to Graphviz, MSCGEN and Microsoft HTML Help Workshop might need to be updated in the .dox files, i.e.:
1. HHC_LOCATION           = "C:\Program Files (x86)\HTML Help Workshop\hhc.exe"
2. MSCGEN_PATH            = "C:\Program Files (x86)\Mscgen"
3. DOT_PATH               = "C:\Program Files (x86)\Graphviz2.38\bin"
